# geowiki v3.1.0

发布日期：2026-07-17

## 语义优先检索

- 基本模式与深度模式均先使用 DeepSeek 完成字段化问题理解、错别字校准和专业转写。
- 检索规划器可将复合问题拆成最多 3 个独立证据目标，并按目标生成仅用于私有知识库的受控查询。
- 基本模式首轮并行执行主查询和证据目标查询；不再因默认文件锚点只检索一个办理环节。
- 模型生成的文件线索只用于本地检索，不作为结论或硬排除条件；用户明确给出的文件范围仍受保护。

## 证据约束

- 对多证据目标问题强制执行证据审查，并仅用审查保留的条款生成最终答案。
- 子查询可以解除系统推定的文件锚点和父问题的 AND 证据组，以召回另一直接法律或技术关系；普通查询排序不受影响。
- 深度研究规划器要求把复合行政问题拆成取得/配置条件与登记/申报程序等独立环节。

## 验证

- 新增复合行政问题的模型规划与跨服务检索计划回归测试。
- 私有知识库实测：夹缝资源配置条件子查询将 `自然资规〔2023〕6号` 排在首位，主查询仍保留 `自然资规〔2023〕4号` 的矿区范围变更登记依据。
- 自动化测试：172 项通过。
