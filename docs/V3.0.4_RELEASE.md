# geowiki v3.0.4 Release Notes

发布日期：2026-07-16

## 修复范围

本版本改进“是否需要评审备案、审批、登记、许可或申请”等行政条件问题的文件优先检索，不以硬过滤方式排除技术标准、办事指南或解读材料。

## 主要变更

- 对行政条件问题补充法规政策候选，避免其被默认技术文档范围排除。
- 对候选采用软排序：优先匹配用户实际行政动作，其次比较上下文条件覆盖度、规范性用语和文件类型。
- 法规政策中的直接条款优先承担“是否需要”的主结论；标准、办事指南和解读材料保留为补充候选。
- 修复“累计查明资源量发生重大变化，是否需要评审备案”被 `DZ/T 0430-2023` 或解读材料抢占首位的问题；`自然资规〔2023〕6号` 第十部分现排在首位。

## 验证

- `PYTHONPATH=src .venv/bin/python -m unittest discover -s tests -q`
- 共 171 项自动化测试通过。
- 本地私有知识库验证：钼矿采矿许可证由自然资源部颁发且采矿期间累计查明资源量发生重大变化的提问，首位候选为 `自然资规〔2023〕6号` 第十部分。

## 数据边界

本版本没有同步或公开真实标准正文、私有知识库、向量索引、应用数据库、`.env` 或凭据。
