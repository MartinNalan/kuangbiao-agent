# geowiki v3.0.2 Release Notes

发布日期：2026-07-16

## 范围

本版本修复矿石加工选冶技术性能试验的等级满足判断，明确区分“等级是否满足”与“试验本身是否符合该等级”。不包含知识库正文、向量索引或部署配置变更。

## 主要变更

- 新增带来源条款的选冶试验等级 Schema：可选性试验、实验室流程试验、实验室扩大连续试验、半工业试验、工业试验。
- 对“半工业试验能否满足实验室流程试验要求”这类问题，默认用户所述试验为对应等级试验，只比较同一等级链，直接输出满足或不满足。
- 新增独立的试验符合性核验意图；样品、设备、处理量、时长、采样和记录要求只在该路径检索，不再附加到等级满足判断。
- 等级关系改由确定性 Schema 计算，避免模型在最终回答阶段补入未陈述的前提。
- 修复“半工业试验”被误识别为“工业试验”的术语重叠问题。
- 修复领域词典背景条目将其他意图的证据约束带入当前查询的问题。

## 验证

- `PYTHONPATH=src .venv/bin/python -m unittest discover -s tests -q`
- 共 167 项自动化测试通过。
- 使用本地私有知识库验证：`半工业试验能否满足详查阶段的实验室流程试验要求？` 被识别为等级满足判断，并从 `DZ/T 0340-2020` 第 5.2.4 条取得等级来源后输出“满足”。
