# Wireframes

## 1. 问答页面

```text
+----------------------------------------------------------+
| geowiki                                                  |
+----------------------------------------------------------+
| [ 输入专业问题，例如：哪个标准规定了金矿基本工程间距？ ] |
|                                             [发送]       |
+----------------------------------------------------------+
| 答案                                                     |
| 根据当前知识库检索结果，...                              |
|                                                          |
| 引用来源                                                 |
| 1. 文件名称 / 条款号 / 页码                              |
|    原文片段...                                           |
| 2. 文件名称 / 条款号 / 页码                              |
|    原文片段...                                           |
|                                                          |
| 检索依据                                                 |
| 全文搜索: 5 条 | 向量检索: 8 条 | 图谱查询: 2 条         |
|                                                          |
| [有用] [不准确] [复制答案]                               |
+----------------------------------------------------------+
```

## 2. 页面元素

- 顶部标题
- 问题输入区
- 答案区
- 引用来源区
- 检索依据区
- 用户反馈区
- 会话历史入口

## 3. MVP 交互

1. 用户输入问题
2. 点击发送
3. 页面显示加载状态
4. 后端返回答案
5. 页面展示答案和引用
6. 用户点击反馈

## 4. 管理员反馈队列

- 展示反馈时间、原问题、原因、补充说明、用户、处理通道和状态。
- `wrong_standard`、`wrong_clause`、`missing_evidence` 标记为知识库审核，不直接修改正式知识库。
- 管理员可更新为待处理、处理中、待 KB 审核、已解决或已忽略，并记录处理说明。
