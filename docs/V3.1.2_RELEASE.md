# geowiki v3.1.2

发布日期：2026-07-18

## 外推对比深度研究修复

- 修复矿体有限/无限外推的跨标准对比在“已召回候选条款”后仍被错误判定为证据不足的问题。
- 外推类型、距离基准、尖推/平推比例和触发条件现在作为同一结构化外推事实的字段处理，不再被模型错误拆成多个必须分别标记的证据目标。
- 锁定外推对比的候选范围、锚点文件、比较维度和受控检索查询，避免深度规划模型改变外推关系的验证合同。
- 每份候选标准最多保留 3 条外推直接候选，以覆盖同一标准内的不同外推情形。

## 验证

- 新增“模型错误拆分外推字段为独立证据目标”的回归测试。
- 自动化测试通过后同步部署至云端。
