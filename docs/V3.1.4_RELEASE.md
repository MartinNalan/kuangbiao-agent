## 采矿权申请材料修复

- 将“办理采矿权需要什么手续”等问法从办理流程路由修正为采矿权申请材料。
- 未明确新立、延续、变更或注销时，先展示四种办理类型供用户选择；不再用父政策的附件概览代替具体材料。
- 办理类型明确后，知识库强制加入自然资规〔2023〕4号附件4的全部对应材料行，并将响应上限提升至 50 条。
- 采矿权材料答案不再引用附件2探矿权申请资料清单。

## 验证

- 本地私有知识库实测：新立申请在默认检索上限下仍返回 14 条附件4材料行，回答不含“附件2”。
- 自动化测试通过后同步部署至云端。
