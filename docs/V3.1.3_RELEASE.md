# geowiki v3.1.3

## 外推事实规范化

- 合并同一标准、同一外推情形下由 OCR 分段产生的“信息不全片段”和“完整片段”，优先保留包含尖推/平推比例的完整事实。
- 保留外推距离基准的正式限定语，例如“相应勘查类型对应的推断资源量工程间距”“拟推资源量类型的经验工程间距”“推断资源量勘查工程间距”。
- 云端私有知识库实测：外推对比从 32 份候选文件中形成 12 条结构化事实，覆盖 10 份文件。

## 验证

- 全量自动化测试通过后同步部署至云端。
